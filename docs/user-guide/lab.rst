===============
Lab Subaccessor
===============
In geotech-pandas, the :class:`~pandas.DataFrame.geotech.lab` subaccessor is simply a namespace
that contains other subaccessors with methods that deal with laboratory tests. In general, the
methods in these subaccessors only handle raw laboratory test data. These subaccessors are further
discussed in the following sections.
